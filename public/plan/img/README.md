# plan/img — 조종 미니게임 SVG 자산

`/play`(수동 조종 미니게임)에서 캔버스에 `drawImage`로 그리는 SVG들입니다.
브라우저는 이 폴더를 `/plan/img/<파일명>` 경로로 서빙합니다.

**파일명을 그대로 유지**하면 됩니다. (경로 매핑은 `app/lib/pilot-sprites.ts`의 `SPRITE_SRC`)
이미지를 교체하려면 아래 같은 이름의 파일을 덮어쓰면 됩니다. 로드에 실패하면
게임은 멈추지 않고 임시 도형(폴백)을 대신 그립니다.

| 파일 | 역할 | 권장 |
|---|---|---|
| `ship.svg` | 조종하는 우주선(마스코트) | 정사각 viewBox, 위쪽이 정면. 충돌 반지름 ≈ 그림의 42% |
| `debris-chip.svg` | 작은 잔해 (가벼움, 2~4kg) | 정사각 |
| `debris-bolt.svg` | 중간 잔해 (5~9kg) | 정사각 |
| `debris-tank.svg` | 큰 잔해 (15~25kg) | 정사각 |
| `hazard.svg` | 위험물 — 닿으면 연료 감소 | 정사각. 붉은 계열 권장 |
| `fuel.svg` | 연료 셀 — 연료 리필/재점화 | 정사각. 청록 계열 권장 |

> 현재 들어 있는 파일은 **플레이스홀더**입니다. 자유롭게 교체하세요.
> 그림 크기(px)는 코드의 `KIND_STAT.drawPx`에서 조절합니다.
