# 📚 「우주 냠냠!」 학습 문서 — 목차

이 폴더는 **프론트엔드를 처음 배우는 사람**이 이 프로젝트의 코드를 완전히
이해하고, 나아가 스스로 고치고 발전시킬 수 있게 되는 것을 목표로 합니다.

모든 문서는 이 프로젝트의 **실제 코드**를 예제로 씁니다. 문서를 읽다가
코드가 나오면, 실제 파일을 열어서 나란히 놓고 보세요. 그게 제일 빨리 늡니다.

> 💡 문서에서는 줄 번호 대신 **파일 이름과 함수 이름**으로 코드를 가리킵니다.
> 줄 번호는 코드를 조금만 고쳐도 어긋나 버리거든요. 에디터에서
> `Ctrl(⌘) + P`로 파일을, `Ctrl(⌘) + F`로 함수 이름을 찾으세요.

## 이 프로젝트의 파일 지도 (요약)

```
app/
├── layout.tsx            ← 모든 페이지의 공통 껍데기 (글꼴, 전역 CSS)
├── page.tsx              ← 첫 화면 "/"
├── globals.css           ← 전역 스타일 (Tailwind, 손그림 상자, 흔들기)
│
├── components/           ← 첫 화면에서 쓰는 부품들
│   ├── doodle-sky.tsx    ← 배경에서 혼자 노는 마스코트 (데모 모드)
│   └── best-score.tsx    ← 최고 기록 표시
│
├── play/                 ← 게임 화면 "/play"
│   ├── page.tsx          ← 페이지 껍데기 (메타데이터 담당)
│   ├── joops-game.tsx    ← ★ 게임 본체 (게임 루프가 여기 있어요)
│   └── game-ui.tsx       ← 점수판·타이틀·게임오버 글자층
│
└── lib/                  ← 게임의 부품 창고 (React를 모르는 순수 로직)
    ├── constants.ts      ← 색·글꼴·종류 등 공용 상수
    ├── doodle.ts         ← ★ 손그림 선 그리기 도구 상자
    ├── backdrop.ts       ← 우주 배경 (별, 달, 모눈 격자)
    ├── debris.ts         ← 우주쓰레기 (데이터 + 움직임 + 그리기)
    ├── mascot.ts         ← 주인공 그리기
    ├── effects.ts        ← 팝업 글자 + 반짝이 입자
    ├── sound.ts          ← 효과음 신시사이저 (mp3 없이 소리 합성!)
    ├── storage.ts        ← 최고 기록 저장/불러오기
    └── canvas.ts         ← 캔버스 해상도 맞추기 (선명하게!)
```

> `docs/archive/` 에는 리팩토링 전 코드 기준의 옛 문서가 보관되어 있습니다.
> 지금 코드와 구조가 다르니 참고만 하세요.
